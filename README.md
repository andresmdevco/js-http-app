# Rick and Morty App

Mini aplicación web construida con **Vite + Vanilla JS** que consume la Rick and Morty API para mostrar episodios aleatorios. 

## ¿Qué hace?

- Obtiene un episodio aleatorio de la Rick and Morty API al cargar
- Muestra el nombre y código del episodio en pantalla
- Permite cargar un nuevo episodio con el botón *Next Episode*

## Tecnologías

- **Vite** — bundler y servidor de desarrollo
- **Vanilla JavaScript (ES Modules)** — sin frameworks
- **Fetch API** — consumo de APIs REST con async/await
- **Rick and Morty API** — https://rickandmortyapi.com

## ⚙️ Instalación y uso

```bash
git clone https://github.com/andresmdevco/js-http-app.git
cd js-http-app
npm install
npm run dev
```

## Conceptos practicados

- Peticiones HTTP con `fetch` y `async/await`
- Manipulación dinámica del DOM (`createElement`, `replaceChildren`)
- Módulos ES (`import` / `export`)
- Manejo de eventos con `addEventListener`
- Funciones asíncronas y Promises (`.then()`)
